# README

ニコリク・ミクノポップエディション

主な使い方

１．NicoRequest.htaを起動します
２．「接続」に、放送番号を入力します。うまくいけば、右上に残り時間が表示されます。
３．リスナーが動画番号を発言すると、自動で動画が「リクエスト」欄に追加されます。
４．主セレを入れたい場合は、「リクエスト」メニューの「テキストの取り込み」でＵＲＬまたは動画番号を入力します。
	いっぺんに入力できます。
５．動画を流すには、再生ボタン（右矢印マーク）を押すか、各動画の「再生」ボタンを押します。
６．自動で流したりランダムに流すこともできます。このあたりはいろいろ遊んでみてください。

あとは、いろいろ遊んでみてください。

細かい使い方

・各動画の動画番号を、
	左クリック：情報を表示
	右クリック：ブラウザで動画ページを開く
・
・

注意点

・枠を延長すると、残り時間は「終了」のままになります。
・ジングルは放送履歴に残りません。
